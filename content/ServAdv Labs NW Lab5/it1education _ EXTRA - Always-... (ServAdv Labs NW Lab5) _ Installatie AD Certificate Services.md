## Installatie AD Certificate Services

##### In deze deelopdracht installeren we de role 'AD Certificate Services' op de server SRV-DC-01.

##### 

##### 

* Log in met de domain administrator op server SRV-MAN-01

> Zorg ervoor dat je de server SRV-DC-01 van hieruit kan configureren.

* Open de Server Manager en kies onder 'Manage' voor "Add Roles and Features"
* Selecteer "Role based or feature based installation"
* Bij het venster 'Select Server' kies je voor SRV-DC-01 en klik je op Next
* Bij het venster 'Server Roles' vink je "Active Directory Certificate Services" aan en klik je op Next
> Bij het pop-upvenster klik je op "Add features".

* Bij het venster 'Features' klik je op Next
* Bij het venster 'AD CS' klik je op Next
* Bij het venster 'Role Services' vink je "Certification Authority" aan en klik je op Next
* Bij het venster 'Confirmation' klik je op "Install"
* Bij het venster 'Results' klik je op "Close" om de wizard af te sluiten
